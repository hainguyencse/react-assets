# admin-assets

### Run story book
```
npm run storybook
```

### Generate static storybook
```
npm run export-storybook
```

### Testing
```
npm run test
```

### Import components
From CLI
```
npm install git+ssh://git@github.com:inspilab/admin-assets.git
```
In code
```
import { Button } from 'admin-assets'
```
